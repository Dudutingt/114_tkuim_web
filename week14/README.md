# Ultralytics Documentation Clone

這是一個模仿 Ultralytics "Models" 文件頁面的靜態網頁專案。本專案透過 HTML、CSS 與 JavaScript 實現了響應式的三欄式版面設計，並復刻了原網站的視覺風格。

## 專案結構

- **index.html**: 網頁的主要結構，包含標頭 (Header)、側邊導航 (Sidebar)、主要內容區 (Main Content) 與右側目錄 (Table of Contents)。
- **style.css**: 負責網頁的樣式設計，包含 CSS 變數定義、Grid/Flexbox 排版、排版樣式與互動效果。
- **script.js**: 處理簡單的前端互動，如 "Ask AI" 按鈕點擊效果、搜尋框互動與主題切換模擬。

## 功能特色

- **三欄式佈局**: 左側模型列表導航、中間主要內容、右側頁內目錄。
- **視覺復刻**: 
  - 使用 Inter 字體與 Ultralytics 品牌藍色 (`#2563eb`)。
  - 擬真的圖表區塊 (SVG 繪製) 與與 "Coming Soon" 標籤樣式。
- **互動元素**:
  - 懸浮的 "Ask AI" 按鈕 (右下角)。
  - 導航列與側邊欄的 Hover 效果。
  - 支援 `Cmd+K` 搜尋快捷鍵模擬。

## 如何執行

1. 將專案的所有檔案 (`index.html`, `style.css`, `script.js`) 下載到同一個資料夾。
2. 直接使用瀏覽器 (如 Chrome, Safari, Edge) 開啟 `index.html` 檔案即可預覽。
